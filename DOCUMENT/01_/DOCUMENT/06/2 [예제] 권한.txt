--------------------------------------------------
chown
--------------------------------------------------
-> 소유권 변경 명령어

-----------------------------------
[환경]
-----------------------------------
mkdir  -p /chown/a
touch /chown/b /chown/a/c
cd /chown
useradd user10

-----------------------------------
[실습]		
-----------------------------------
chown user10:user10 /chown		- /chown 의 소유자,소유그룹변경
chown root.root /chown			- /chown 의 소유자,소유그룹변경
chown user10 /chown			- /chown 의 소유자만 변경
chown root /chown			- /chown 의 소유자만 변경
chown .user10 /chwon			- /chown 의 소유그룹만 변경
chown -R user10.user10 /chown		- /chown 의 하위파일&디렉토리 포함 변경

--------------------------------------------------
chmod
--------------------------------------------------
-> 허가권 변경 명령어

-----------------------------------
[환경]
-----------------------------------
mkdir 	/chmod
touch 	/chmod/a
cd	/chmod


-----------------------------------
허가권 부여방식
-----------------------------------
--------------------
numeric 
--------------------
->숫자 형식으로 부여

-----------
확인
-----------
chmod 755 a
chmod 566 a
chmod 755 a

--------------------
symbolic
--------------------
->문자 형식으로 부여
 		
-----------
symbolic(문자) 방식
-----------
u+,- rwx : 소유권자 권한변경 
g+,- rwx : 소유그룹 권한 변경
o+- rwx  : other 권한 변경     
a +,- rwx  : 모두 권한 변경

-----------
확인
-----------
chmod u-rw a	
chmod g+w a
chmod o+rw a
chmod a-x a
chmod u+x,g+x a

참고)여러 개의 권한 변경시 콤마(,) 를 이용하여 붙여서 쓴다.


-----------
미니문제
-----------
rw_r_xr__	->
rwxr__r_x	->
r__rwxr__	->
r_xr__r_x	->
r__r_x___	->

755	->
644	->
701	->
542	->
643	->
375	->



--------------------------------------------------
실습 - 권한 확인 
--------------------------------------------------

-----------------------------------
[환경]
-----------------------------------
useradd user1
useradd user2
useradd user3
passwd user1
passwd user2
passwd user3
mkdir /permission
cat > /permission/testfile
This is testfile!
^C
chmod 751 /permission
chmod 640 /permission/testfile
chown -R user1.user2 /permission


-----------------------------------
[User별 권한 확인]
-----------------------------------
ls -ld /permission 

--------------------
1. 디렉토리
--------------------
user1 접속
cd 명령어 - ok(x권한)
touch 명령어 - ok(w권한) 
ls 명령어 - ok(r권한)

user2 접속
cd 명령어 - ok
touch 명령어 - x 
ls 명령어 - ok

user3 접속
cd 명령어 - ok
touch 명령어 - x 
ls 명령어 - x

권한해제  : chmod 555 /permission
--------------------
2. 파일
--------------------

user1 접속
cat 명령어 - ok
vi 명령어 - ok 

user2 접속
cat 명령어 - ok
vi 명령어 - x

user3 접속
cat 명령어 - x
vi 명령어 - x
