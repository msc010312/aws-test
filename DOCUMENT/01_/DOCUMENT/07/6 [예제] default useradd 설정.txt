--------------------------------------------------------
/etc/default/useradd
--------------------------------------------------------

----------------------------------------
1) vi /etc/default/useradd로 변경
----------------------------------------


----------------------------------------
2) useradd -D 명령어로 변경
----------------------------------------

--------------------------
[옵션]
--------------------------
-s : 기본쉘변경
-b : 기본 홈디렉토리 변경

--------------------------
[실습]
--------------------------
useradd -D -s /bin/ksh

useradd test200
cat /etc/passwd

useradd -D -b /export/home 

useradd test201
cat /etc/passwd

★/etc/default/useradd 를 다시 원래설정으로 변경하세요
--------------------------------------------------------
[미니문제]
--------------------------------------------------------
userdel -r test1
userdel -r test2
userdel -r test3

1.
명령어를 이용해서 홈디렉토리의 기본경로를 /export/hometest 로 변경하고
test1 계정을 만드세요

2.
명령어를 이용해서 쉘의 기본경로를 /bin/sh로 변경하고
test2 계정을 만드세요

3. 
vi를 이용해서 쉘의 기본경로를 /bin/bash, 홈디렉토리 경로를 /home으로 되돌리고
test3 계정을 만드세요
