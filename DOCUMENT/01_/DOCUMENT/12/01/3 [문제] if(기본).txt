입력값 두개를 받아서 두값이 같다면 
"참입니다"

다르다면 
"거짓입니다"

를 출력하세요

파일명 : if1.sh


------------------------------------
답
------------------------------------
#!/bin/sh
echo "두수를 입력하세요"
read num1
read num2

if [ $num1 = $num2 ]
then
echo "참입니다"
else
echo "거짓입니다"
fi
exit 0
~
