인자 2개($1,$2)를 받아서 수가 같다면

'두수는 같은 수입니다'
 S1가 S2보다 크다면
'$1가 $2보다 큽니다'

$1가 $2 보다 작다면
'$1가 $2보다 작습니다'  라는 문장을 출력하도록 하세요

파일명 : if2.sh

---------------------------------------------------------
답)
---------------------------------------------------------
/bin/sh
if [ $1 -eq $2 ]
then
echo "두 수는 같은 수 입니다"
else
        if [ $1 -gt $2 ]
        then
        echo "$1 가 $2 보다 큽니다"
        else
                if [ $1 -lt $2 ]
                then
                echo "$1 가 $2 보다 작습니다"
                fi
        fi
fi
exit 0
